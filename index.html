Bot Features

Track your members' all time punishments with /record!

Staff Commands

  • report - Submits a report to the staff's logs channel.
  • ban - Restricts a user's access to the server.
  • unban - Removes a user's banned status earlier.
  • baninfo - View details about a banned user.
  • kick - Kicks a certain user out of the server.
  • mute - Restricts a user from sending messages.
  • unmute - Removes a user's muted status earlier.
  • muteinfo - View details about a muted member.
  • warn - Sends a warning message to a user.
  • record - Displays a member's punishment history.
  • giverole - Adds a role to a user.
  • takerole - Deletes a role from a user.
  • addnote - Adds an admin note on someone's account.
  • viewnotes - Views all notes linked to an account.
  • clear - Bulk deletes a certain amount of messages.
  • suggestionchannel - Sets a channel for suggestions to be sent in.
  • suggestion - Accept or decline a suggestion from your suggestion channel.
  • remindme - Sets a timer for a reminder.
  • checknames - Check a member's previous nicknames.

View one or multiple people's avatar(s) with /avatar!

Info Commands

  • help - Displays a list of all available commands.
  • serverinfo - Displays information about the server.
  • userinfo - Outputs information about a user's account.
  • avatar - Displays the avatar(s) of certain users.

Get logs whenever someone deletes/edits a message!

Logging Commands

  • setlogschannel - Sets a custom logging channel.
  • togglemsglogs - Toggles message logs on/off.

Send messages when a member joins/leaves the server!

Welcome Commands

  • setwelcomechannel - Sets a custom welcome channel.
  • welcomerole - Sets a role to be assigned to new users.
  • welcomemessage - Sets a custom welcome message.
  • welcomedm - Sets a custom welcome DM.
  • togglewelcomedm - Toggles welcome DMs on/off.
  • setleavechannel - Sets a custom leave channel.
  • leavemessage - Sets a custom goodbye message.
  • toggleleavemsg - Toggles leave messages on/off.

Get real time weather information with /weather!

Fun Commands

  • weather - Tells you information about the weather.
  • define - Looks up a term in the dictionary.
  • dogfact - Sends a lovely dog fact.
  • catfact - Same as dogfact, except it's for cats.
  • nasanews - Looks up a term on NASA's website.

Change your server's prefix with /setprefix!

Debug Commands

  • bugreport - Submits a bug report to the bot's server.
  • botsuggestion - Submits a suggestion to the bot's Discord server.
  • setprefix - Changes the "/" prefix to a custom one.
  • ping - Displays the bot's current latency in ms.
  • invitelink - Sends the invite link for the bot.
  • discordserver - Sends an invite link to the bot's server.
  • serversettings - Sends current bot settings for this server.
